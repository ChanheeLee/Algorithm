# Study Repository 이용안내

- Github Desktop 다운 경로 : https://desktop.github.com/

- 자신의 아이디로 로그인 후, Algorithm 저장소 Clone

- CodeByWeek에 주차 별 폴더에 자신의 영문이름으로 디렉토리 생성

- 코드는 '문제번호-사용한 언어' 파일명 메모장 형식으로 업로드 해주시기 바랍니다.

- commit 하기 전 반드시 최신 파일들을 fetch한 후 업로드 해주시기 바랍니다.(desktop버전은 sync버튼)

- 업로드 후, 저한테 Pull requests를 보내주시면 conflict 안나게 관리하겠습니다.

- 자세한 이용방법은 카톡으로 문의주시기 바랍니다.
